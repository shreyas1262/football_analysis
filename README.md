# football_analysis
A learn-by-building football analytics project covering the full modern data and AI engineering stack. Ingests match data from football-data.org via Apache Airflow, transforms it through a medallion architecture using dbt and PostgreSQL, then layers an AI tier on top — including a Claude-powered Q&A agent with tool calling, RAG over match reports via pgvector, natural language to SQL, auto-generated match narratives, anomaly detection, and a weekly insight email. Built without frameworks like LangChain — every pattern implemented from first principles using the Anthropic SDK directly.
